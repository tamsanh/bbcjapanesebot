BBC Japanese Bot
================

BBC Japanese Bot is a bot that automatically posts to the /r/BBCJapanese
subreddit. It's made by /r/magicpurpledinosaur and was inspired by the bot made
by /r/NHKEasyNews.

Dependencies
------------

* Python 3
    * feedparser
    * Requests
    * lxml
        * cssselect
    * html2text
